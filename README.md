# openhim-mediator-shell-script
OpenHIM Mediator for executing shell scripts.

## Usage

## Development
Checkout from GitHub
```
git clone https://github.com/jembi/openhim-mediator-shell-script.git
npm install
```
To run the mediator in development mode
```
grunt serve
```
or to run the test
```
npm test
```
